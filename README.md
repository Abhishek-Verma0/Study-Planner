# Smart Study Planner

A responsive web-based **Smart Study Planner** built using **HTML, CSS, and Vanilla JavaScript** to help students organize subjects, manage schedules, track tasks, and monitor study progress.
The application uses **LocalStorage** to ensure all data persists even after page refresh or browser restart.
